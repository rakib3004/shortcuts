# Basic commands for Angular CLI

## Install all packages
```bash
npm install
```
## Run angular project
```bash
ng serve
```
## Run & Open angular project
```bash
ng serve --open
```
## Create New Component
```bash
ng g c user-component
```
## Create New Module
```bash
ng g m user-module
```
## Create New Service
```bash
ng g s user-service
```
